## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Backend API running on http://localhost:5000

### Installation

```bash
npm install
```

### Development Server

```bash
npm start
```

The app opens at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

Creates optimized production build in `build/` folder.

## API Configuration

The frontend connects to the backend API via proxy configured in `package.json`:

```json
"proxy": "http://localhost:5000"
```

All API calls are made to `/api/*` endpoints which are proxied to `http://localhost:5000/api/*`

## Authentication

- JWT tokens stored in localStorage
- Tokens included in all API requests via headers
- Token expires in 30 days
- Auto-login on page refresh if token exists

## Folder Organization

### Components
Reusable, single-responsibility components:
- **EmptyState** - Display when no data
- **JobCard** - Job listing card
- **InterviewCard** - Interview card
- **JobForm** - Add/Edit job form
- **Navbar** - Navigation header
- **StatsCard** - Statistics display

### Pages
Full page components with layouts:
- **LoginPage** - User login
- **DashboardPage** - Main dashboard
- **JobListPage** - All jobs
- **JobDetailsPage** - Single job details
- **AddJobPage** - Create new job
- **InterviewSchedulePage** - Schedule interview
- **ProfilePage** - User profile

### Context
Global state management:
- **JobContext** - Jobs, interviews, updates state

### Services
API integration:
- **jobSearchService** - External job search API

### Utils
Helper functions:
- **csvUtils** - CSV export/import
- **notificationUtils** - Notification helpers

## Environment Setup

The frontend is configured to proxy requests to the backend. Ensure:

1. Backend is running on `http://localhost:5000`
2. Backend has CORS enabled
3. API endpoints match backend routes

## Build & Deployment

### Local Build
```bash

```

### Deploy to Vercel
1. Push code to GitHub
2. Connect Vercel to repo
3. Set environment variable: `REACT_APP_API_URL=https://your-api.com`
4. Deploy

### Deploy to Netlify
1. Build locally: `npm run build`
2. Drag `build/` folder to Netlify
3. Set redirect rules for SPA routing

## Troubleshooting

### Port 3000 Already in Use
```bash
npm start -- --port 3001
```

### Can't Connect to Backend
- Check backend is running: `http://localhost:5000`
- Check CORS is enabled in backend
- Check proxy in `package.json`

### Blank White Page
- Press F12 to see console errors
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Check localhost:3000 in address bar

### Build Size Too Large
- Run: `npm run build`
- Analyze: `npm install -g source-map-explorer`
- Then: `source-map-explorer 'build/static/js/*.js'`
